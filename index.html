    Twitch Bot ✨

    ❓ A simple Twitch channel bot created on my live streamings for helping and interacting with my viewers/chat.

    📋 Requirements

    You can check the requirements for this project on the requirements page.

    🔎 Usage

    1. Clone the repo to your machine.
    2. Update or install dependencies:

    poetry install

    1. Setup your .env with your Twitch channel data, you can use the .env.example as a template.
    2. Change your commands messages or create new ones
    3. Start the project:

    poetry run bot 🎉 | Successfully connected to the channel @amandamartinsdev on Twitch.
